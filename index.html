My first Game Jam with Lundum Dare 50

I took part in my first Ludum Dare 50 Game Jam. With my team we wanted to do a game where the light slowly fades out and the player has to keep it lit up going through a maze with monsters. For most of us, it was our first time using Unity, we had very little skills in C#, and it added itself to a busy schedule (school work...), so we couldn't finish the game, but it still was a very cool project and fun experience.

My Unity Game made for The CNAM-ENJMIN application

To apply to the CNAM-ENJMIN, the school I wanted to join, I had to create a unity project inspired from boids, so here it is, it's a game where you have to make your ant civilisation survive as long as possible. The "boids" part are the ant colony, which follows the player and act as boids.

Minecraft server hosted on a Raspberry Pi 4

After running a Raspberry on Raspbian, then using it as an arcade console with RetroPie, I installed a Minecraft server running on a Raspberry Pi 4 model B. To do this, I installed Linux on a SD card, retrieved resources from GitHub, used and modified a Bash code and managed an installation via ssh.
